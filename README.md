# СИБКОМПЛЕКТ — AI Ассистент

Умный чат-бот для подбора электрооборудования и автоматизации коммерческих предложений.

## Стек

- **Backend**: Python 3.11+, FastAPI, uvicorn
- **AI**: YandexGPT (Yandex Foundation Models API)
- **Каталог**: ETM iPRO API (ipro.etm.ru)
- **Парсинг документов**: pdfplumber, Pillow
- **Frontend**: HTML5 + CSS3 + Vanilla JS (встроен)

## Быстрый старт

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Настроить переменные окружения

```bash
cp .env.example .env
```

Отредактировать `.env`:

```env
# ETM API (ipro.etm.ru)
ETM_LOGIN=ваш_логин
ETM_PASSWORD=ваш_пароль

# Yandex Cloud — YandexGPT
YANDEX_API_KEY=ваш_api_key
YANDEX_FOLDER_ID=ваш_folder_id
YANDEX_GPT_MODEL=yandexgpt-lite   # или yandexgpt для более мощной модели
```

**Где получить Yandex Cloud API ключ:**
1. Зайти на https://console.yandex.cloud
2. Создать сервисный аккаунт с ролью `ai.languageModels.user`
3. Создать API-ключ для сервисного аккаунта
4. Скопировать `folder_id` из настроек облака

### 3. Запустить

```bash
python main.py
```

Открыть в браузере: http://localhost:8000

## API эндпоинты

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/` | Чат-интерфейс |
| GET | `/api/session` | Создать новую сессию |
| POST | `/api/chat` | Отправить сообщение |
| POST | `/api/upload` | Загрузить ТЗ (PDF/PNG/TXT) |
| POST | `/api/generate_kp` | Сформировать КП |
| GET | `/api/session/{id}/tz` | Получить распознанное ТЗ |
| GET | `/api/session/{id}/kp` | Получить КП |
| GET | `/api/health` | Статус системы |

## Демо-режим

Если API ключи не настроены, бот работает в демо-режиме:
- Парсинг ТЗ возвращает пример данных по КТП
- КП генерируется с условными ценами
- Чат отвечает информационным сообщением

## Функциональность

### Загрузка документов
- PDF (текстовые и сканированные)
- Изображения: JPG, PNG, WEBP, GIF
- Текстовые файлы TXT
- Drag & drop на любую область страницы

### Парсинг ТЗ
YandexGPT анализирует документ и извлекает:
- Список оборудования с параметрами
- Количество, единицы измерения
- Адрес доставки
- Примечания

### Поиск в ETM
По каждой позиции ТЗ выполняется:
- Поиск по артикулу / наименованию
- Получение цены (с/без НДС, розничная)
- Проверка наличия на складах
- Загрузка характеристик и изображений

### Генерация КП
Команда «Сформировать КП» создаёт:
- Позиционированную спецификацию
- Цены за единицу и итого
- Сроки поставки
- Итоговые суммы с/без НДС
- Условия доставки и оплаты
